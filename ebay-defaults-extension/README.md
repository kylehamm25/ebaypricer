# eBay Listing Defaults

Chrome extension that fills listing form defaults with one click.

## Install

1. Open `chrome://extensions`
2. Enable **Developer mode**
3. Click **Load unpacked**
4. Select the `ebay-defaults-extension` folder

## Usage

- Click the **Apply Defaults** button on any eBay listing page
- Or click the extension icon and press **Apply Defaults**
- Edit defaults via the extension popup

## Customize

Open the popup and click "Edit defaults" to change values without touching code.
